CSS nth-child Pseudo-Class Demonstration

The :nth-child() pseudo-class allows you to select elements based on their position inside a parent element. Below are examples with visual demonstrations and corresponding code snippets.

1. Selecting the Third Child

  • Item 1
  • Item 2
  • Item 3 (Selected)
  • Item 4
li:nth-child(3) {
    background-color: red;
    color: white;
}

2. Selecting Even & Odd Children

  • Item 1 (Odd)
  • Item 2 (Even)
  • Item 3 (Odd)
  • Item 4 (Even)
li:nth-child(odd) {
    background-color: lightgreen;
}

li:nth-child(even) {
    background-color: lightblue;
}

3. Selecting Every 3rd Child

  • Item 1
  • Item 2
  • Item 3 (Selected)
  • Item 4
li:nth-child(3n) {
    background-color: orange;
}

4. Selecting First & Last Child

  • First Item (Blue)
  • Middle Item
  • Last Item (Purple)
li:first-child {
    background-color: blue;
    color: white;
}

li:last-child {
    background-color: purple;
    color: white;
}

5. Selecting the Third Last Child

  • Item 1
  • Item 2
  • Item 3
  • Item 4 (Selected)
li:nth-last-child(3) {
    background-color: pink;
}
